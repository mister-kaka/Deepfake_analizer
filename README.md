# Deepfake_analizer
С нуля реализованная архитектура CNN которая выявляет дипфейки фото, легковесная(14 миллионов параметров)

Deepfake Detection Project — Команда №6
Репозиторий содержит проект по разработке системы бинарной классификации для обнаружения дипфейков. 

Авторы
Кирилл Соколов — Проектирование и реализация архитектуры EfficientNet с нуля, оптимизация пайплайна обучения, проведение экспериментов.
Мария Начинова — Исследование базовых CNN-решений, EDA и подготовка первичного датасета.

Технологический стек: PyTorch, torchvision, Pandas, NumPy, Scikit-learn, Matplotlib
Архитектурные решения: Реализация блоков MBConv, использование Depthwise Separable Convolutions, механизмы Squeeze-and-Excitation.

Ключевые особенности реализации
1. Собственная реализация EfficientNet
Вместо использования стандартных библиотечных моделей, архитектура была написана вручную на PyTorch.
Это позволило:
Глубоко контролировать параметры масштабирования (width, depth, resolution).
Самостоятельно реализовать инвертированные остаточные блоки.
Интегрировать блоки внимания для улучшения селективности признаков.
2. Борьба с дисбалансом и оверфиттингом
Sampling: Реализован взвешенный загрузчик данных
Augmentation: Применен кастомный пайплай наугментаций (повороты, изменение экспозиции, отражения)
3. Анализ весов и инференс
Разработан инструментарий для загрузки и проверки состояния модели (state_dict).
Реализована визуализация структуры слоев и весовых коэффициентов для контроля корректности обучения.
